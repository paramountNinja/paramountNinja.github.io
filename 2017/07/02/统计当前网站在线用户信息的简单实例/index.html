统计当前网站在线用户信息的简单实例

发表于 2017-07-02 更新于 2020-02-17

“本例基于Web基础的servlet展开，虽然较简单，但对Web的三大组件进行了结合。 ”

主要实现三个功能：

  • 提供用户登录界面，提供注销功能。
  • 显示当前网站的所有登录用户信息
  • 管理员可以踢除指定的登录用户

友情提示： Web的三大组件都需要在web.xml中配置哦！
